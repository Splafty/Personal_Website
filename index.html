XML

XML (eXtensible Markup Language) is a hierarchical data format using tags to define elements. It allows for nested and structured data. It's human-readable and suitable for complex data structures.

CSV

CSV (Comma-Separated Values) is a simple, text-based format where each line represents a record, and fields are separated by commas. It's easy to read and widely supported but may struggle with complex structures.

JSON

JSON (JavaScript Object Notation) is a lightweight, text-based format for data interchange. It uses key-value pairs and supports nested structures. It's widely used in web development and easy for both humans and machines to read.
